# ✂️🪨📄 Rock-Paper-Scissors Game

A command-line Rock-Paper-Scissors game built using Python.

The player competes against the computer, and the game determines the winner based on standard Rock-Paper-Scissors rules.

---

# ✨ Features

* 🎮 User vs Computer Gameplay
* 🎲 Random Computer Selection
* 🏆 Win/Lose/Draw Logic
* 📊 Score Tracking
* 🔄 Play Multiple Rounds
* ⚡ Easy-to-Use Interface

---

# 🛠️ Technologies Used

* Python 3
* Random Module
