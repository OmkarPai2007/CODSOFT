## 🧮 Calculator

A simple command-line calculator that performs basic arithmetic operations.

### Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⚠️ Division by Zero Handling
* 🔄 Multiple Calculations Support

### Features
