# 🔐 Password Generator

A Python application that generates strong and random passwords based on user-defined length.

This project helps users create secure passwords using letters, numbers, and special characters.

---

# ✨ Features

* 🔢 Custom Password Length
* 🔠 Uppercase & Lowercase Letters
* 🔣 Special Characters
* 🔐 Strong Random Passwords
* ⚡ Fast Password Generation

---

# 🛠️ Technologies Used

* Python 3
* Random Module
* String Module
